# Dream Data — strona (GitHub Pages)

## Jak uruchomić lokalnie
Najprościej w VS Code:
1) Zainstaluj wtyczkę **Live Server**
2) Kliknij prawym na `index.html` → **Open with Live Server**

## Publikacja na GitHub Pages
1) Wrzuć cały folder projektu do repozytorium (np. `dreamdata-strona`)
2) GitHub → **Settings** → **Pages**
3) Source: **Deploy from a branch**
4) Branch: `main` + folder `/root`
5) Zapisz — po chwili dostaniesz link do strony

## Dane, które edytujesz bez grzebania w kodzie
- `team.json` — zespół + zdjęcia w `assets/team/`
- `linkedin-feed.json` — aktualności (mock lub z automacji)
- `site.json` — sekcja Kontakt (email/telefon/linki/teksty)
